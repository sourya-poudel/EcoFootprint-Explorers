# EcoFootprint Explorers

A fun, interactive web application that helps users calculate and understand their personal carbon emissions — covering transport, home energy, food, waste, and awareness. Users can also take a quiz, view leaderboard results and pledge to plant a tree.

## Demo

Check it out at: [https://eco-footprint-explorers.vercel.app](https://eco-footprint-explorers.vercel.app)
The main sections include: Calculator, Quiz, Leaderboard, My Results, and Pledge a Tree.

## Features

* A **Carbon Footprint Calculator**: users input data on travel, electricity/home usage, food habits, waste & recycling, and awareness items to compute an estimated footprint.
* **Quiz**: engage users with questions that raise awareness of sustainable behaviour.
* **Leaderboard**: track and compare results with other users.
* **My Results**: view your past entries and progress.
* **Pledge a Tree**: take action by pledging to plant a tree or join an eco-event.
* Responsive UI, designed for both desktop and mobile.
* Aimed at increasing awareness and motivating sustainable changes.

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/eco-footprint-explorers.git
   cd eco-footprint-explorers
   ```
2. Install dependencies:

   ```bash
   npm install
   # or
   yarn install
   ```
3. Configure environment variables (if any):

   ```bash
   # .env.local
   NEXT_PUBLIC_API_URL=...
   ```
4. Run locally:

   ```bash
   npm run dev
   ```

   The app will be available at `http://localhost:3000`.

## Usage

* Navigate to the **Calculator** to enter your data and compute your carbon footprint.
* Take the **Quiz** to test your awareness and earn points.
* Visit the **Leaderboard** to see how you rank.
* Go to **My Results** to review your past submissions.
* Click **Pledge a Tree** to commit to planting a tree or joining an eco-event.
* Encourage others to use the app and compare their results!
